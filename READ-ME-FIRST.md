- **Issues** = Tasks that live in repositories
- **Projects** = Dashboards that DISPLAY issues

Issues DON'T move. Projects LINK to them.

Think:
- Repository = Filing cabinet (where documents live)
- Project = Dashboard (shows filtered view)
