## Delegation Best Practices

### When Delegating:
- [ ] Clear task description
- [ ] Specific deadline
- [ ] Create GitHub issue
- [ ] Add to this tracker
- [ ] Set follow-up reminder
- [ ] Confirm team member has capacity

### Follow-up Checklist:
- [ ] Check progress on scheduled date
- [ ] Update status in tracker
- [ ] Remove blockers if any
- [ ] Adjust deadline if needed
- [ ] Recognize completed work
