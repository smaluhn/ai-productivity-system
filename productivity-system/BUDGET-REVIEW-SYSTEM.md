## Weekly Review Checklist

### Monday Morning (9:00 AM)

**Step 1: Data Collection (5 mins)**
- [ ] Open personal bank accounts
- [ ] Open Favourse bank accounts
- [ ] Open expense tracking apps
- [ ] Gather receipts from last week

**Step 2: Update Spreadsheets (5 mins)**
- [ ] Log personal income
- [ ] Log personal expenses
- [ ] Log Favourse income
- [ ] Log Favourse expenses

**Step 3: Analysis (5 mins)**
- [ ] Compare to budget
- [ ] Identify variances
- [ ] Check income source progress
- [ ] Calculate runway

**Step 4: Action Items (3 mins)**
- [ ] List issues to address
- [ ] Set goals for this week
- [ ] Update forecasts
- [ ] Schedule follow-ups

**Step 5: Documentation (2 mins)**
- [ ] Update weekly review note
- [ ] Add to daily note
- [ ] Share with relevant people (accountant, co-founder)
